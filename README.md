# Como rodar o projeto (frontend)

- Clonar o repositório (``git clone https://github.com/Matiaszz/frontend-blog-mindgroup.git``)
- ``cd frontend-blog-mindgroup``
- Criar um arquivo ``.env`` em ``frontend-blog-mindgroup``, substituindo os placeholders pelos valores reais.
- Instalar dependências (``npm i``)
- Rodar o projeto (``npm run dev``)
